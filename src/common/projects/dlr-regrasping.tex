\cventry
{Master Thesis project in collaboration with DLR Robotics and Mechatronics Center}
{Development of Regrasping Framework for Robotic Manipulation}
{Deutsches Zentrum für Luft- und Raumfahrt e.V. (DLR), Oberpfaffenhofen (DE)}
{Apr 2025 - Dec 2025}
{
    Research project focused on improving manipulation reliability through intelligent regrasping strategies, validated in high-fidelity simulation prior to hardware deployment on DLR lightweight robotic platforms.
    \newline
    \vspace{4mm}
    \begin{cvitems}
        \item {Developed regrasping algorithms for multi-stage manipulation tasks, achieving \textbf{33\% higher success rate} compared to baseline single-grasp approaches in benchmarking evaluations.}
        \item {Built calibrated \textbf{digital twin} in \textbf{PyBullet} physics simulator, accurately modeling DLR lightweight robotic arm kinematics, hybrid compliant gripper dynamics, and workspace constraints.}
        \item {Implemented complete \textbf{simulation-to-deployment pipeline}, validating manipulation strategies in simulation before transferring to real hardware with minimal tuning.}
        \item {Integrated mechanical interfaces, vision sensors, and control algorithms within \textbf{ROS-based} software architecture for modular task execution.}
        \item {Conducted systematic performance analysis across different object geometries, grasp configurations, and environmental conditions to identify failure modes and optimization opportunities.}
    \end{cvitems}
}
