\cventry
{Academic project for Data Analysis for Mechatronics Systems course}
{SHM as a Multivariate Outlier Detection Problem}
{Politecnico di Milano, Milan (IT)}
{May 2025 - May 2025}
{
    Statistical approach to damage detection in civil structures under environmental variability, using unsupervised learning techniques to distinguish damage from benign operational changes.
    \newline
    \vspace{4mm}
    \begin{cvitems}
        \item {Analyzed vibration data from tie-rod elements subjected to controlled damage scenarios and environmental variations (temperature, humidity).}
        \item {Applied \textbf{Mahalanobis Squared Distance (MSD)} and \textbf{Principal Component Analysis (PCA)} to extract damage-sensitive features from multivariate sensor data.}
        \item {Established damage detection thresholds using statistical control charts, quantifying false alarm rates under normal operating conditions.}
        \item {Evaluated robustness of different statistical indices to environmental confounders, identifying optimal feature sets for reliable damage identification.}
    \end{cvitems}
}
