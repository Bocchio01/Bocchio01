\cventry
{Educational project for teaching electromechanical principles}
{Transparent DC Motor Model for Educational Demonstrations}
{Personal Workshop, Como (IT)}
{Feb 2023 - Feb 2023}
{
    Functional demonstration model built to visually explain electromagnetic induction and motor operation principles to students.
    \newline
    \vspace{4mm}
    \begin{cvitems}
        \item {Designed and constructed visible motor assembly using salvaged lawn mower components and custom wooden supports for educational clarity.}
        \item {Enabled hands-on exploration of brushed DC motor concepts including commutation, torque generation, and speed control.}
        \item {Used as teaching aid to bridge theoretical electromagnetics with practical machine design for peer study groups.}
    \end{cvitems}
}
