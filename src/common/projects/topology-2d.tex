\cventry
{Academic project for Topology Optimization course}
{Topology Optimization of 2D Structures with CONLIN Algorithm}
{Politecnico di Milano, Milan (IT)}
{Nov 2024 - Dec 2024}
{
    Implementation of gradient-based optimization routines from first principles, applying the Convex Linearization (CONLIN) method to structural topology problems.
    \newline
    \vspace{4mm}
    \begin{cvitems}
        \item {Developed optimization framework in \textbf{MATLAB}, implementing sensitivity analysis and design variable updates using CONLIN approximations.}
        \item {Validated algorithm on benchmark problems including discrete topology selection and continuous material distribution optimization.}
        \item {Implemented convergence criteria and filtering techniques to ensure mesh-independent solutions and prevent checkerboard patterns.}
        \item {Analyzed algorithm performance across different problem formulations, comparing convergence rates and computational efficiency.}
    \end{cvitems}
}
