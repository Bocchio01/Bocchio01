\cventry
{Personal project to experiment with ROS and AI/ML techniques}
{Design and implementation of a self-driving RC car}
{Personal Workshop, Como (IT)}
{Feb 2025 - PRESENT}
{
    Modular software architecture inspired by \textbf{Autoware}, integrating high-level perception and planning on a Linux-based SBC (\textbf{Raspberry Pi}) with real-time actuator control on a microcontroller.
    \newline
    \vspace{4mm}
    \begin{cvitems}
        \item {Implemented a low-level \textbf{C++} firmware supporting multiple kinematic models (Ackermann \& Differential) and smooth acceleration profiles.}
        \item {Developed a modular \textbf{ROS 2} stack featuring custom packages for trajectory following, command arbitration (Gate logic), and human-machine interaction.}
        \item {Integrating \textbf{LiDAR-based SLAM} and \textbf{IMU} sensor fusion for real-time localization, mapping, and navigation in indoor environments (currently under development).}
    \end{cvitems}
}
